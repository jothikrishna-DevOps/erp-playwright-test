# Frontend UI

Next.js React application with calm, minimal design inspired by Isha Foundation.

## Features

- Dashboard with test list
- Record new test page
- Real-time status updates
- Calm, earthy color scheme
- Responsive design

## Pages

- `/` - Dashboard (test list)
- `/record` - Record new test

## Design Philosophy

- **Calm**: Soft backgrounds, generous spacing
- **Minimal**: Clean typography, no clutter
- **Earthy**: Earth and sage tones
- **Simple**: Clear actions, intuitive flow

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Environment Variables

```env
NEXT_PUBLIC_API_URL=http://localhost:3001
```
